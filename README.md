# Kivy Learning

## Introduction

    This is a self learning project which is going to serve as a playground and learning environment for Kivy Cross Platform Development
    library. This project will be divided as follows:

    1. Layouts
        This section deals with all common layouts used in kivy by implementing them programmatically or in kv design file.
        
    2. Widgets
        This section deals with all common widgets, their properties, relationship and their manipulation.

    3. Canvas
        This section deals with base of the application i.e. an environment where all the layouts and widgets are placed and correlate to each other in order to function as an application

    4. Projects
        This section deals with custom made projects based on this learning experience.
